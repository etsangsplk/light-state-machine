# Light State Machine

Simple state machine for different purposes. It comes with an example uri parser implementation that uses the state machine to parse URI's.

## System requirements

    Linux
    Mingw
    C++11 g++ or clang

## Compile

Build:

    make

Release:

    make RELEASE=on

Verbose:

    make VERBOSE=on

Clean:

    make clean

Tests:

    make tests
